# trazza

Web local para controlar el resultado real de cuentas de prop firms: compras, resets, activaciones, fees, payouts y refunds.

## Uso

Abre `index.html` directamente en el navegador.

Los datos se guardan en el navegador con `localStorage`. Puedes exportar una copia en JSON desde la propia app.

## Incluye

- Dashboard con resultado neto, gastos, retiros, ROI, break-even y cuentas activas.
- Registro de firms.
- Registro de cuentas.
- Registro de movimientos economicos.
- Filtros y tablas por firm, cuenta y movimiento.
- Exportacion JSON/CSV e importacion JSON.
